






To download the folder or repo
Command: git clone "Paste the <> Code"

After some changes:
1) Command: git init
Either initilize or reinitialize (in the case of previous existence)

2) git status
To see the changes in the project
if shown in red colour = some changes has been made
if showin in green colour = no changes has been made

3) git add .  
(Add all the files for upload)\

4) git commit -m "text to describe changes"

5) git pull 
To pull or be updated about the changes made by other 
6) git push -u origin master
